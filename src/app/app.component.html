Files Splitter

This will allow you to split a lot of files to chunks each chunk in a folder i.e(split big folder into subfolders). Base on your configuration. Path is the absolute path of the folder that contains the files. Max number of files is how many files each subfolder should contains. Size of subfolder is approximate number that determine each subfolder size. Note: It's approximate because files are vary in size! So, when the program add each file it will check if the subfolder exceeds the size then it will stop.

Note: You should determine max number of files or size of subfolder or both. You can't leave them empty. If you determine both then the first one reach will apply.
